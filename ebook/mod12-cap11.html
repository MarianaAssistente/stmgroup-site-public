Gerando Imagens com Gemini Imagen 4

# Script para gerar imagem com Gemini Imagen 4 cat > ~/.openclaw/workspace/shared/mediagen.sh << 'EOF' #!/bin/bash PROMPT="$1" OUTPUT_NAME="$2" OUTPUT_DIR="$HOME/shared/media" mkdir -p "$OUTPUT_DIR" curl -s -X POST \ "https://generativelanguage.googleapis.com/v1beta/models/imagen-4.0-fast-generate-001:generateImages?key=$GEMINI_API_KEY" \ -H "Content-Type: application/json" \ -d "{ \"prompt\": {\"text\": \"$PROMPT\"}, \"number_of_images\": 1 }" | python3 -c " import json, sys, base64 data = json.load(sys.stdin) img_bytes = base64.b64decode(data['generatedImages'][0]['image']['imageBytes']) open('$OUTPUT_DIR/$OUTPUT_NAME.png', 'wb').write(img_bytes) print('Imagem salva: $OUTPUT_DIR/$OUTPUT_NAME.png') " EOF chmod +x ~/.openclaw/workspace/shared/mediagen.sh # Uso: bash shared/mediagen.sh \ "laptop em ambiente escuro com luz dourada, fundo minimalista, estilo premium" \ post_instagram_capa
🚨
NUNCA use hex colors no prompt do Gemini Imagen. Em vez de #D4AF37, escreva "cor dourada brilhante". Em vez de #7c3aed, escreva "roxo profundo vibrante". Códigos hex causam comportamento imprevisível ou são ignorados pela API.
🚨
Google Drive URLs são rejeitadas pelo Meta. Para publicar imagens no Instagram via Meta Graph API, você precisa de uma URL pública direta — não do Drive. Use Imgur como CDN gratuito: faça upload da imagem no Imgur e use a URL direta i.imgur.com/xxx.jpg no parâmetro image_url da API.

Criando Vídeos com fal.ai

# Exemplo: gerar Reel com Seedance v1.5 Pro (fal.ai) curl -s -X POST \ "https://fal.run/fal-ai/seedance-v1-5-pro" \ -H "Authorization: Key $FAL_API_KEY" \ -H "Content-Type: application/json" \ -d '{ "prompt": "Empreendedor usando laptop em escritório moderno, luz dourada lateral, movimento suave da câmera, estilo cinematográfico", "duration": 5, "aspect_ratio": "9:16" }' # Avatar falante com Hedra Character-3 (foto + áudio → vídeo) # 1. Gerar áudio com ElevenLabs # 2. Enviar foto + áudio para Hedra # 3. Receber vídeo com lip sync realista
💡
Caso de uso prático: Use Gemini Imagen para gerar a imagem do post → ElevenLabs para narração → Creatomate para adicionar legendas → Meta Graph API para publicar no Instagram. Tudo automatizado via cron job. Isso é o que seu agente Digital faz toda semana.
