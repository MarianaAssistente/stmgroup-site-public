Capítulo 03

LLMs & OpenRouter: Escolhendo Seu Modelo

Seu agente precisa de um "cérebro" — um modelo de linguagem que entende e gera texto. Neste capítulo você vai aprender a configurar modelos gratuitos via OpenRouter, e quando vale a pena migrar para pagos.

O Que é um LLM?

💡
LLM em 1 linha: Large Language Model — um modelo de IA treinado em bilhões de textos, capaz de entender e gerar linguagem humana. É o "cérebro" por trás do ChatGPT, do Claude e do Gemini. Sem um LLM, seu agente não consegue nem dizer "olá".

Para o seu agente funcionar, ele precisa de um LLM disponível via API. A boa notícia: você não precisa pagar nada para começar. Em 2026 existem modelos gratuitos e de altíssima qualidade disponíveis via OpenRouter.

💡
O que é uma API? API é uma "janela de serviço". Em vez de abrir o ChatGPT no navegador, seu agente "bate na janela" da API e recebe a resposta de volta. Pense no garçom que vai à cozinha buscar seu prato — a API é esse garçom.

OpenRouter: O Marketplace de LLMs (GRÁTIS para começar)

🛒
O que é o OpenRouter? Um marketplace com +200 modelos de IA numa única API. Com uma única chave você acessa Claude, GPT, Llama, Gemini, Mistral e muito mais. E o melhor: vários modelos são completamente gratuitos.

Como criar sua conta no OpenRouter (5 minutos)

1

Acessar openrouter.ai e criar conta

Abra openrouter.ai no navegador → clique em "Sign Up" → cadastre com Google ou e-mail. É gratuito, não precisa de cartão de crédito para usar os modelos free.

2

Gerar sua API Key

Após entrar, clique no menu → Settings → API Keys → Create Key. Dê um nome (ex: "meu-agente"), clique em Create. Copie a chave gerada — ela aparece só uma vez! Ela vai parecer com: sk-or-v1-xxxxxxxxxxxx

🚨
API Key = senha. Nunca compartilhe publicamente. Se vazar, revogue imediatamente em Settings → API Keys → Delete e crie outra.
3

Encontrar modelos GRATUITOS

Acesse openrouter.ai/models → no filtro superior, clique em "Free". Só modelos gratuitos aparecem. Você também pode identificá-los pelo sufixo :free no nome. Todos funcionam sem saldo na conta.

Configurando no OpenClaw — Dois Caminhos

Agora que você tem sua API key, é hora de configurar. Você tem dois caminhos — escolha o que preferir:

Via Telegram (mais fácil)

Após o agente estar ativo, simplesmente mande uma mensagem:

— Conversa no Telegram — Você: configure o modelo para llama-3.3-70b gratuito do openrouter Agente: ✅ Configurando... Modelo atualizado: meta-llama/llama-3.3-70b-instruct:free Provider: OpenRouter Custo estimado: R$0/mês (modelo gratuito) Testando conexão... ✅ OK! Você: ótimo! pode me dizer qual modelo está usando agora? Agente: Estou rodando com meta-llama/llama-3.3-70b-instruct:free via OpenRouter. É gratuito e tem excelente qualidade para uso geral. 🧠

Via TUI (controle total)

1

Abrir o OpenClaw na VPS

No terminal da sua VPS, digite openclaw e pressione Enter. A interface TUI (Terminal UI) vai abrir com um menu visual.

2

Navegar até Settings

Use as teclas de seta ↑↓ para navegar. Selecione "Settings" e pressione Enter. Depois selecione "Models".

3

Configurar o Provider

Em "Provider", selecione "OpenRouter". Um campo para API Key vai aparecer — cole a sua chave (sk-or-v1-xxx).

4

Definir o modelo principal

Em "Primary model", digite: meta-llama/llama-3.3-70b-instruct:free e pressione Enter para salvar.

Via arquivo openclaw.json (direto)

Se preferir editar o arquivo de configuração diretamente:

# Abrir o arquivo de configuração nano ~/.openclaw/workspace/openclaw.json
~/.openclaw/workspace/openclaw.json
{ "workspace": "/home/usuario/.openclaw/workspace", "models": { // Modelo gratuito como principal — ótimo para começar! "default": "meta-llama/llama-3.3-70b-instruct:free", // Fallback também gratuito "fallback": "google/gemini-2.0-flash-exp:free" }, "providers": { "openrouter": { // Sua API Key do OpenRouter (openrouter.ai/settings) "apiKey": "SUA_API_KEY" } }, "channels": { "telegram": { "token": "SEU_TOKEN_TELEGRAM" } } }

Quando Vale a Pena Usar Modelos Pagos?

Modelos gratuitos são excelentes para aprender e para uso leve. Quando suas automações forem críticas — ou você quiser mais qualidade de raciocínio — considere pagar. Mas não tem pressa: comece do zero, evolua com a demanda.

💰 Quanto vai custar por mês?

Mês 1 (aprendendo): R$0 — use apenas modelos gratuitos.
Mês 3 (uso moderado): R$20–80 — se migrar para pagos.
Mês 6 (automações intensas): R$55–130 com VPS incluída — ainda muito menos que qualquer assistente humano ou ferramenta no-code.
