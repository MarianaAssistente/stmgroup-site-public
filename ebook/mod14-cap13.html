Estratégias de Otimização

1. Use Modelos Hierárquicos

Nem toda tarefa precisa do melhor modelo. Configure prioridades:

~/.openclaw/workspace/openclaw.json
{ "models": { // Tarefas simples: gratuito "default": "anthropic/claude-4-haiku-20250501", // Análise média: barato "analysis": "openrouter/google/gemini-flash-1.5", // Tarefas críticas: premium "critical": "openrouter/anthropic/claude-4-haiku-20250501" } }

2. Cache de Respostas

Para tarefas que repetem muito (FAQ, respostas padrão), salve as respostas em arquivo e evite chamar a API toda vez:

Crie um arquivo cache/faq.json no workspace com as perguntas e respostas mais frequentes que recebo. Use o formato JSON com perguntas como chaves e respostas como valores. Inclua exemplos genéricos que eu possa editar depois.

3. Limitar Tarefas de Alto Custo

Monitore quais tarefas consomem mais tokens e ajuste a frequência:

# Verificar uso de tokens no OpenRouter # Acesse: openrouter.ai/activity # Veja quais modelos e endpoints estão consumindo mais # Dica: relatórios longos consomem muitos tokens # Limite para 1x por dia, não a cada hora

Quando Fazer Upgrade da VPS?

💡
Sinais de que está na hora de fazer upgrade: Agentes demorando mais de 5 segundos para responder, servidor travando com múltiplas tarefas simultâneas, ou quando tiver 5+ agentes rodando ao mesmo tempo. O upgrade é simples: 1 clique no painel da Hostinger (hPanel).
Cenário VPS Recomendada Custo Aprox.
1-2 agentes, uso pessoal 1 vCPU, 1GB RAM R$25/mês
3-5 agentes, pequena empresa 2 vCPU, 4GB RAM R$80/mês
10+ agentes, escala comercial 4 vCPU, 8GB RAM R$200/mês
