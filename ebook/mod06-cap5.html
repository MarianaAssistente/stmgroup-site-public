Capítulo 05

Conectando ao Telegram

O Telegram é o canal perfeito para interagir com seu agente de IA. É gratuito, tem API robusta, funciona no celular e no computador, e você não precisa de nenhum servidor extra para usar.

Por Que Telegram?

💡
Vantagens do Telegram: Diferente do WhatsApp (que exige número de telefone verificado e tem restrições severas de API), o Telegram oferece API gratuita, ilimitada e sem burocracia. Em 5 minutos você tem um bot funcional.
💡
Por que Telegram e não WhatsApp? O WhatsApp exige número de telefone verificado e tem restrições severas de API. O Telegram tem API gratuita, ilimitada e sem burocracia. Em 5 minutos você tem um bot funcional — sem aprovação, sem espera.
ℹ️
Nota: O OpenClaw suporta outros canais além do Telegram (WhatsApp, Discord, Slack, Signal e mais). Para este guia, usamos apenas o Telegram — o mais simples e direto para começar. Quando quiser explorar outros canais, a documentação do OpenClaw tem guias específicos para cada um.

Criando Seu Bot no Telegram

1

Abrir o BotFather

No Telegram, pesquise por @BotFather (é o bot oficial do Telegram para criar bots). Clique em "Start" ou envie /start.

2

Criar um novo bot

Envie o comando /newbot para o BotFather. Ele vai pedir o nome do bot (qualquer nome) e depois o username (deve terminar em "bot", ex: meu_assistente_bot).

3

Copiar o Token

Após criar, o BotFather vai enviar uma mensagem com o token HTTP API. Parece algo assim: 7123456789:AAHFxxxxxxxxxxxxxx. Copie e guarde esse token.

🚨
Token = senha do bot. Com esse token, qualquer pessoa pode controlar seu bot. Nunca compartilhe publicamente. Se vazar, revogue imediatamente com /revoke no BotFather.

Configurando o Token no OpenClaw

⚠️
Esta é a última configuração pelo terminal. Depois disso, o Telegram estará ativo e você usará o TUI ou o próprio Telegram para qualquer outra configuração.
# Última edição pelo terminal — após isso, use o TUI nano ~/.openclaw/workspace/openclaw.json

Atualize a seção channels.telegram.token:

~/.openclaw/workspace/openclaw.json
{ "workspace": "/home/usuario/.openclaw/workspace", "models": { "default": "anthropic/claude-4-haiku-20250501" }, "providers": { "openrouter": { "apiKey": "sk-or-v1-SUACHAVEAQUI" } }, "channels": { "telegram": { // Token que você recebeu do BotFather "token": "7123456789:AAHFxxxxxxxxxxxxxx" } } }

Iniciando o Agente

# Ir para a pasta do agente cd ~/.openclaw/workspace # Iniciar o OpenClaw (ele vai ler o openclaw.json automaticamente) openclaw start # Você deve ver uma mensagem como: # ✅ OpenClaw iniciado # 🤖 Agente conectado ao Telegram: @meu_assistente_bot # 🧠 Modelo: anthropic/claude-4-haiku-20250501

Testando o Bot

Abra o Telegram, pesquise pelo username do seu bot (ex: @meu_assistente_bot), clique em Start e envie uma mensagem de teste.

💡
Primeira mensagem teste: "Olá! Quem é você e o que você faz?" — O agente vai responder com base no que está no SOUL.md. Se a resposta fizer sentido, está funcionando!

A partir de agora: Esqueça o Terminal

Com o Telegram configurado, você tem controle total do seu agente — direto no celular. Não precisa mais abrir SSH, terminal ou qualquer ferramenta técnica.

Tudo que você pode fazer pelo Telegram:

  • 💬 Conversar com o agente normalmente, em português
  • 📁 Pedir para criar arquivos, relatórios, análises
  • 🔔 Receber relatórios e alertas automáticos no celular
  • 🔗 Configurar novas integrações ("Conecte meu Gmail")
  • ⏰ Criar automações ("Crie um lembrete todo dia às 8h")
  • 🤖 O agente responde exatamente como no TUI — mas no seu celular, em qualquer lugar
Regra de ouro: Qualquer coisa que você pediria por SSH, você pode pedir pelo Telegram. O agente tem os mesmos poderes nos dois lugares — só muda o canal.
